# angular2-webpack-seed
A starter kit featuring Angular 2, TypeScript, and Webpack by [AngularClass](https://angularclass.com)

### install
```bash
$ npm webpack -global
$ npm webpack-dev-server -global
$ npm install
```

### server
```bash
$ webpack-dev-server
```

### auto build files
```bash
$ webpack --watch
```
